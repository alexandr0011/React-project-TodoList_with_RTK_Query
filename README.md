# TodoList App

## Приложение "Список дел" написаное на React с использованием RTK Query, хранение и взаимодействие со списком дел на сервере mockApi, кастомная пагинация
# Technologies:
- React 
- Redux Toolkit
- RTK Query
- MockAPI
- Lottie-react
- Module-scss
- Pagination
